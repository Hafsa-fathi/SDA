Main Objectives

The main objective is to come up with an effective solution to solve the major problems of driver by developing a smart driver assistant mobile application software with intelligent voice assistant using artificial intelligence, deep learning, machine learning and convolutional neural network. As an application software that should be applied as an alerting system for avoiding the accidents due to the drowsiness and careless of road signs. Application should be an assistant for the driver which should be reminding the pinned locations in the map when the user reaching the certain radius of the destination. Moreover, the mobile application should recognize the voice/command of the user and performs the needs to the driver/user.

Sub Objectives

In order to achieve the main goal of this proposed system has successive sub-objectives should be completed.
• User can add the multiple destinations through voice in the Map. And user can access/ control the system through voice.
• Alert the user before a certain radius of distance (0-1000m) to each pinned destination as a notification where the user to be perform a task.
• Display the locations to the user in the Device with pinned Destinations.
• Improve the Accuracy level of Speech to Text and Text to Speech conversions.
• Detecting all available parking slots using Image processing techniques.
• Creating parking slots detector.
• Save the vacant slot details in firebase.
• Evaluate and test the developed application to see if it meets the user's criteria or not, to ensure customer satisfaction.
• Fixing raspberry pi 3 model B+, raspberry pi camera module 5MP, Mobile application and power supply.
• Design a system to control the brake and activate an alarm via Arduino.
• Training the machine learning model by using different algorithms, then select the best suitable algorithm with high accuracy.
• Implement graphical user interfaces and system logic based on the system specifications.
